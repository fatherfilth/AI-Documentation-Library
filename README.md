# AI Documentation Library

Centralized knowledge base for AI models, tools, skills, repos, agents, and conversation logs.

## Structure

```
docs/
  models/       # AI model documentation
  tools/        # Tool and integration docs
  skills/       # Skill definitions and guides
  repos/        # Repository reference docs
  agents/       # Agent configurations and specs
  _index/       # Cross-reference indexes
  _templates/   # Document templates
conversations/
  2026/         # Conversation logs by year
```
